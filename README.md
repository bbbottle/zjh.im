## ZJH.IM

zjh.im render zjh's [personal site](https://zjh.im).

## Install

`npm install @bbbottle/zjh.im` or `yarn add @bbbottle/zjh.im`
## Javascript API
```javascript
renderZjhDotIm($dom)
```
